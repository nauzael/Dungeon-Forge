HOW THE FIX WORKS
=================

Problem:  OAuth login hangs forever
Cause:    100ms reload too fast for Supabase
Solution: 2000ms delay (2 seconds) before reload

Timeline:
  Google OAuth → Deeplink (instant)
  → Extract tokens (instant)
  → WAIT 2 SECONDS ← Supabase processes tokens
  → Reload page
  → Supabase processes valid tokens
  → onAuthStateChange fires with session
  → User logged in ✓
