STEP 4: Build APK (1 minute)
===========================

    build-and-deploy.bat

    bash build-and-deploy.sh

    ./gradlew assembleDebug

The APK will be at:

STEP 5: Install on Device (30 seconds)
======================================

Make sure device is connected and USB debugging is enabled.

Run:

    adb install -r android/app/build/outputs/apk/debug/app-debug.apk

You should see:
    Success

WHAT WAS FIXED
==============

Problem:
    OAuth login hangs forever - user sees loading screen indefinitely

Root Cause:
    Page reloaded in 100ms, before Supabase processed OAuth tokens

Solution:
    Wait 2000ms (2 seconds) before reloading

Code Location:
    File: App.tsx
    Line: ~90
    Change: setTimeout(..., 2000) // was 100ms

Result:
    Supabase has time to process tokens, session is valid, app works ✓
