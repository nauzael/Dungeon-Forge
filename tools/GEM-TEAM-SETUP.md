## 📋 Qué es Gem-Team

Gem-Team es un **plugin de IA para planificación y gestión de equipos** que ayuda a:
- 📊 Organizar tareas y features
- 👥 Asignar trabajo a miembros del equipo
- 🎯 Crear y gestionar sprints
- 📈 Trackear progreso

---

## 🚀 Cómo Usar Gem-Team

### Opción 1: VS Code + Cursor AI
El repositorio contiene plugins para:
- `.claude-plugin/` - Plugin para Claude
- `.cursor-plugin/` - Plugin para Cursor AI
- `.apm/` - Atom Plugin Manager
- `.opencode-plugin/` - OpenCode Plugin

### Opción 2: CLI (Command Line)
Consulta el README en `tools/gem-team/README.md`

### Opción 3: Documentación
Lee `tools/gem-team/README.md` para instrucciones completas

## 🎯 Próximos Pasos

### Para usar Gem-Team en tu proyecto:

1. **Lee la documentación:**
   ```bash
   cat tools/gem-team/README.md
   ```

2. **Si usas VS Code + Cursor/Claude:**
   - Abre la carpeta `.claude-plugin/` o `.cursor-plugin/`
   - Sigue las instrucciones de instalación

3. **Para planificar multiclases:**
   - Usa gem-team para crear un plan de sprints
   - Asigna tareas del PLAN-MULTICLASS-IMPLEMENTATION.md
   - Trackea progreso
