CAMBIOS ESPECÍFICOS DEL FIX
==========================

ARCHIVO: App.tsx

Cambio #1 - IMPORT (Línea 24):
  const OAuthDebugConsole = lazy(() => import('./components/OAuthDebugConsole'));
  
  Qué hace: Importa el panel de debug de OAuth

Cambio #2 - TIMEOUT CRÍTICO (Línea 92):
  }, 2000);  // ← ANTES era 100
  
  Qué hace: Espera 2 segundos antes de recargar
  Por qué: Supabase necesita tiempo para procesar los tokens OAuth
  
Cambio #3 - LOGGING (Línea 70+):
  console.log('[Login] Starting Google OAuth...');
  console.log('[OAuth] Deeplink received...');
  console.log('[OAuth] Found tokens extracted:', !!tokens);
  console.log('[OAuth] Setting window.location.hash');
  console.log('[OAuth] Waiting 2 seconds before reload...');
  
  Qué hace: Registra cada paso del proceso
  Por qué: Permite ver exactamente dónde está el problema

Cambio #4 - COMPONENTE RENDER (Línea 709):
  <Suspense fallback={<div>Cargando...</div>}>
    <OAuthDebugConsole />
  </Suspense>
  
  Qué hace: Muestra el botón 🔍 en la UI
  Por qué: Permite monitorear OAuth sin necesidad de ADB

=============================

Ver logs en tiempo real mientras pruebas:

Ver todos los logs de la app:
  adb logcat tag:*:V | grep "dungeforge"

Capturar logs a archivo:
  adb logcat > oauth-logs.txt
  # ... prueba OAuth aquí ...
  # Ctrl+C para parar

Limpiar logs antes de prueba:
  adb logc
